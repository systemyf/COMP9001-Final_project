# COMP9001-Final_project
WatermarkSeal
WatermarkSeal: 图片水印添加工具（保护你的图片）
WatermarkSeal是一个水印添加程序，用于给图片添加水印。用户可以使用该项目，很方便的为单张图片，或者多张图片添加水印，使得图片在网络上可被追踪或仅公开部分信息。与常见程序不同的是，该项目允许用户手动选择水印添加的内容、位置、大小，可以像盖章一样为图片添加文字水印。同时也可以选择自动平铺整个页面添加水印或者遮盖。该程序使用到了python的tkinter库作为图形界面，使用到了PIL库作为图片处理库，同时用到了矩阵处理运算，类，函数，循环等多种python编程知识。

## 运行

您可以运行以下命令来启动 WatermarkSeal 桌面端水印与图片浏览器界面：

```bash
python watermark_seal.py
```
